guppy-FAQ　2001-Mar-09

Ｑ：カタログがリストに表示されない
Ａ：2つ以上のカタログがあるときは、一番始めのカタログが表示されます。一覧表示の上部にあるメニューで他のカタログを選択できます。ここに表示されていない場合は、スクリプトの中で、addcatalog()で登録する必要があります。なにも登録されていないときにも、addlaneで参照されたカタログは自動登録されます。

Q：カタログリストの表示を変更したい。
Ａ：カタログの中に、formatという項目を作り、ここに関数定義を文字列で記入します。これには、関数を記述して、２重の[[ ]]でかこみます。
[[ function (rec) .... end ]]

Q：2つのカタログを結合するには
Ａ：mbCatalogUnion()を使います。ただし、２つのカタログに重複がない場合は、mbCatalogJoin() のほうが高速です。mbCatalogSubset()では集合の差をとることもできます。

Q：addlane()でレーンを追加してもなにも表示されない。
Ａ：必要なパラメータがぬけているか、selectionでの選択で、何も選択できなかったかです。selection={} とするとすべて選択されます。

Q：カタログはどこまで自由につくれるのか。
Ａ：各項目における属性値は　symbol,pos1,pos2は必ず必要です。label,pen,fill,sphape　が指定されていると、表示に使われます。masterは、元になったカタログを指定します。これらを他の用途に自由に使うことはできません。アンダースコアで始まる属性名は使うべきではありません。その他は制限はなく、何でも作れます。データ型は数値、文字列、テーブルなど何でもかまいません。テーブルを指定した場合は、saveでの保存はされません。

Q：使える色は。
Ａ：数値で1-30が使えます。32-63は、グラデーションになっています。
(1...11) YIvBLUE YIvRED YIvMAGENTA YIvGREEN YIvCYAN YIvYELLOW YIvWHITE YIvSILVER YIvKARKI YIvPINK YIvORANGE
(17...30) YIvIVORY0 YIvBLACK0 YIvNAVY0 YIvGRAY0 YIvLIME0 YIvAQUA0 YIvCREAM0 YIvOAK0 YIvCURSOR0 YIvWHITE0 YIvPERU0 YIvTEAL0 YIvMOS0 YIvMAROON0

Q：使えるプリミティブは。
Ａ：
"GPYbox"、"GPYthinbox"、"GPYarrow"、、。現在は縦の高さが変更できません。

Q：plotはどうするのか。
Ａ：現在はプロトタイプの状態ですが、プリミティブGPYplotとGPYcbarがあります。
カタログとして下記のようなwaveデータを用意します。xのstep は1000がデフォルトです。
mydata={ wave={1,1,2,2,1,1,2,2} }

レーンごとのラベルを指定したり、文字の大きさを変えたい。
細かい制御はできませんが、カタログのテーブルにlabel属性をつけるとsymbolでなくてその文字で表示します。また、カタログ全体にlabel属性をつけるとすべてその文字で表示します
